#  Weather & AQI Tracker

A Python-based command-line weather dashboard that fetches real-time weather conditions and Air Quality Index (AQI) for any city worldwide, with a local search history feature.

#  How to run on your own device

**1. Clone the repository**
**2. Install Python.**
**3. Make and env file with you own api key in it**
**4. Run the .py file**
